AI Agent Layer — Claude & Gemini on GitHub

💬 @claude on Issues/PRs

Mention @claude on any issue or PR and it starts working — writes code, commits progress, asks for feedback. Up to 30 turns per invocation.

👁️ Auto PR Review

Every PR is automatically reviewed by Claude on open and sync. Skips bot PRs and release-please PRs. Runs concurrently.

🔧 Auto-Fix Failures

When CI fails, Claude analyzes logs, applies fixes for lint/type/format errors, and opens issues for design problems. Flood guard limits to 2 open fix PRs.

📐 AI Rules Sync

Canonical rules in .rulesync/ are auto-synced to Claude Code, Gemini, Copilot, and Cursor configs. One source of truth for all AI tools.

🧬 git-repo-agent — Async Repository Quality Maintenance

A custom Claude Agent SDK application that autonomously maintains repository health across all projects. It runs independently of the normal development cycle — onboarding new repos, scoring health across 5 categories (docs, tests, security, quality, CI), auto-fixing safe issues, and diagnosing pipeline failures by correlating kubectl, ArgoCD, GitHub Actions, and Sentry data. Seven specialized subagents (blueprint, configure, quality, security, docs, test-runner, diagnose) handle different aspects. The key value: repositories stay healthy even when no one is actively working on them.

Onboard new repos Health scoring Auto-fix safe issues Pipeline diagnostics 7 specialist subagents
